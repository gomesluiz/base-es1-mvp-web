# 🚀 Laboratório de Engenharia de Software Ágil com IA (Docs-as-Code)

Bem-vindo ao template base para o desenvolvimento de software moderno. Aqui, você não escreve código sem antes entender o negócio. Utilizaremos Inteligência Artificial não para "burlar" o aprendizado, mas como uma **ferramenta de orquestração arquitetural**.

## 🗺️ A Trilha de Desenvolvimento (4 Etapas)
1. **Etapa 1:** [Lean Inception](/docs/01_lean_inception/)
2. **Etapa 2:** PBB e BDD (O Contrato Técnico) - `/docs/02_pbb/`
3. **Etapa 3:** Prototipação e Código (A Sprint 1) - `/src/` e `/tests/`
4. **Etapa 4:** DevOps e Nuvem (O Grand Finale) - `Dockerfile` e `.github/workflows/`

## 🛡️ Regras Inegociáveis (`.cursorrules`)
Este repositório contém uma "Constituição" para a IA (`.cursorrules`). Ele força a IA a criptografar dados sensíveis de moradores e proíbe a geração de código espaguete. **Leia este arquivo antes de começar.**
